# Drowsy-Driver-Safety-Kit
Monitors eye, head &amp; steering wheel movements. If the driver falls asleep, an SOS message is sent to the emergency contact.

#### Components 

1. Raspberry pi-2

2. GSM

3. GPS

4. IR Sensor

5. Accelerometer

6. Buzzer

7. Crash Sensor

8. Gyroscope Sensor

9. DC Motor
